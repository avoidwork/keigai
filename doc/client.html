<static> request(uri, type, args, headers) → {Object}

Creates an XmlHttpRequest to a URI ( aliased to multiple methods )

The returned Deferred will have an .xhr

Parameters:
Name Type Description
uri String

URI to query
type String

[Optional] Type of request ( DELETE/GET/POST/PUT/PATCH/HEAD/OPTIONS ), default is GET
args Mixed

[Optional] Data to send with the request
headers Object

[Optional] Custom request headers ( can be used to set withCredentials )

} );
