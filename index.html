keigai

keigai is a lightweight data store library, providing a powerful model & reactive UI components, with 1-n relationships.

keigai data stores have MVCC inspired record versioning, for an easy "undo", & they can be wired to APIs.

keigai can be installed from npm & bower, and supports AMD loaders or script tags.

API documentation is included, and available at http://api.keigai.io
